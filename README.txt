STEPS TO RUN THIS SHINY APP DASHBOARD

1- Open the `Folder 1 Agaba` and open `micode Agaba.Rproj`

2- In the Opened `micode Agaba.Rproj` OPEN the `app.R` File by going to the folder `Shiny Agaba` > `shiny` > `app.R`
2b- Change the "source([type the File-path for `Data_Cleaned.R` file here])"

3- Then in the Opened `micode Agaba.Rproj` OPEN the `Data_Cleaned.R` File by going to the folder `Folder 1 Agaba` > `Data_Cleaned.R`
3b- Change the "setwd([type the directory for `Folder 1 Agaba` folder here])"

4- Run the `app.R` to view the Dashboard
